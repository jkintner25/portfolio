Pic Picker

The inspiration for this app came from a coding competition/challenge hosted by a discord server. The challenge was to build and host a web app that allows users to view images which they can add or remove to/from their collection of favorites. We were required to use localStorage to accomplish this. I built my app using JavaScript + React.js + Next.js + Tailwind.CSS.

Notable features include utilizing the useReducer React Hook and the Unsplash API to 1)pull recent images and 2)search images based on keyword(s).

Eventzeit

This web app was inspired by Eventbrite. I had the pleasure of working with three colleagues to bring the project to completion. One of my duties was implementing Google Maps API to render a map centered on the event location, displayed on each event details page. I also built out and integrated the functionality associated with purchasing, displaying, and returning tickets.

About

I'm a full-stack software engineer living in Birmingham, AL. As a former dog professional, I've discovered that mechanisms for problem-solving in training dogs are very similar to those inherent in building web apps—. It's such a rewarding process of challenge => success => challenge => success.

I'm currently employed by Tricorder Array Technologies where I am the lead frontend developer. We're a startup building the technology to allow researchers the ability to better understand and care for the needs of their research animals. Because we are a small company, I'm not only building frontend UIs, I'm managing databases, creating APIs, and building out our devops routines and infrastructure. There's never a dull moment and I'm continually learning. I incorporate Agile methodologies and processes like Feature-Driven Development and Stand-Ups .

I'm absolutely thrilled to be in this field and can't wait to see all of the problems I'll participate in solving.
